# lukin uniapp 开发微信小程序

### 前言

lukin 是一款微信小程序

本项目使用 uniapp 开发，为实现一端开发多端兼容

### 技术栈

```js
uniapp
uview - ui
sass
```

### 安装项目依赖

```js
npm install // 安装前端依赖
cd server
npm install // 安装后端依赖
```

### 运行项目

```js
hbuilder点击运行到想要的环境
```
